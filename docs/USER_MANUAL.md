# SkillTrack Pro — User Manual

This manual explains how to use the SkillTrack Pro mobile application.

## What SkillTrack Pro does

SkillTrack Pro helps teams manage internship / learning programs:

- Admin creates programs, assigns mentors, and enrolls learners.
- Admin/mentor creates milestones and tasks (program-level).
- Learners view tasks, submit work, and receive feedback.
- Mentors review submissions and track learner progress.
- Notifications inform users about submissions, reviews, and deadlines.

## Login

1. Open the app.
2. Enter your email and password.
3. Tap **Login**.

After login, you are redirected based on your role:

- Learner → Learner Dashboard
- Mentor → Mentor Dashboard
- Admin → Admin Dashboard

## Notifications

- Tap the **bell icon** in the top-right of your dashboard.
- The red badge shows the number of **unread** notifications.
- Tap a notification to open its details.
- Unread notifications become read after you close the details popup.

Notifications typically appear when:

- A learner submits a task.
- A mentor reviews a submission.
- A deadline reminder is generated by the backend.

# Learner Guide

## Learner Dashboard

From the dashboard you can:

- See **Pending Tasks**, **Approved Tasks**, and **Completion %**.
- Open **Performance Report**.
- Browse and open **Active Programs**.
- Open notifications and logout.

## Browse Programs

1. From the dashboard, tap **Browse all my programs**.
2. Tap any program to open the program detail.

## Program Detail (Tasks)

A program contains milestones and tasks. Tasks belong to the program.

1. Open a program.
2. You will see the task list.
3. Tap a task to open task details.

## Task Detail

You can:

- Read the task description and deadline.
- Copy resource links (if provided).

### Submit your work

1. Open a task.
2. Scroll to **Submission**.
3. Enter your submission link (GitHub/Drive/etc.)
4. Add optional notes.
5. Tap **Submit**.

### Submission rules

- After you submit once, the submission becomes **locked**.
- You can still view your submitted link and notes.
- If you need changes after submission, contact your mentor/admin to handle it.

### Feedback and score

After mentor review, you may see:

- Status (approved/rejected)
- Score (if provided)
- Feedback text

## Performance Report

- Open from the dashboard.
- Shows approved/rejected/pending counts and average score.

# Mentor Guide

## Mentor Dashboard

From the mentor dashboard you can:

- See **Pending reviews** (submissions waiting for review).
- Open the submissions inbox.
- View assigned learners and open each learner timeline.
- View your programs.
- Open notifications and logout.

## Review Submissions

1. From mentor dashboard, tap **Review submissions**.
2. Filter by status:
   - Submitted (pending)
   - Approved
   - Rejected
3. Tap a submission to open the review screen.

### Review screen behavior

- By default, the review screen is **view-only**.
- Tap **Edit** (top-right) to enable editing.
- Choose decision (Approve/Reject), enter score (0–100), and add feedback.
- Tap **Save review**.

## Learner Timeline

1. From mentor dashboard, tap an assigned learner.
2. The timeline shows submissions + review events.

If timeline is empty:

- The learner may not have submitted anything yet.
- Pull down to refresh.

## Mentor Programs

1. Open **View my programs**.
2. Tap a program to open overview.

# Admin Guide

## Admin Dashboard

From admin dashboard you can:

- Manage users
- Manage programs
- View audit logs
- Open notifications and logout

## Manage Users

You can:

- View users by role
- Create users (admin/mentor/learner)

## Manage Programs

Programs are the main container for milestones and tasks.

Typical admin workflow:

1. Create a program.
2. Assign mentor to the program.
3. Assign learners to the program.
4. Create milestones and tasks.

### Program Detail

Inside a program, you can:

- Assign learners (dropdown)
- Change mentor (dropdown)
- Create milestone
- Create task

### Creating tasks

- Tasks belong to the program.
- Learners enrolled in the program automatically see those tasks.
- Deadline selection is via calendar + clock (date/time pickers).

## Audit Logs

Audit logs help track important actions (who did what and when).

# Common Troubleshooting

## I cannot see any programs/tasks

- Ask admin to assign you to a program.
- In local development, ensure backend seeding is done (`npm run seed`).

## Notifications badge is not updating

- Pull to refresh the dashboard.
- Open notifications and come back; the dashboard reloads.

## Network errors

- Check that the backend server is running.
- Ensure the app is using the correct API base URL for your device type.
